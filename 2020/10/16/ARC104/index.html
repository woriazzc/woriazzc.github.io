C - Fair Elevator

 

题意：在 1 到 2n 的数轴上给定 $n$ 个区间，定义一组合法的区间集合：所有端点恰被使用一次，交叉的区间长度必须相同。给定的区间中有 -1 表示数据缺失，问给定的区间集合是否有可能为合法的区间集合。

区间dp

合法区间的形式显然是如上图这样的。要么是能分成左右两部分都是合法区间，要么自己是一个最小的合法区间。

最小的合法区间：区间长度为偶数，且左半区间只有左端点，右半部分只有右端点，且所有区间长度都为总长度的一半。

 

D - Multiset Mean

 

题意：给定n，k，1 到 n 每个数可以选 0 到 k 次，对于 $1\le x\le n$，求凑出集合，集合均值为 x 的方案数。

dp

均值显然需要转化。

所以就是要选出一些数，和相等。可以看到左侧 $a_i-x\in[1,n-x]$，右侧 $x-a_j\in [1,x-1]$。

$dp[i][j]$ 表示用 1 到 i，和为 j 的方案数。

枚举和 s，把 $dp[n-x][s]\cdot dp[x-1][s]$ 计入答案。

下面考虑怎么计算dp。

如果没有 $k\le K$ 这个限制的话，$dp[i][j]=\sum dp[i-1][j-ki]$，也就是等于所有与 $j$ 模 $i$ 同余的 dp 之和。所以可以维护一个ps[]，记录模 $i$ 同余的等价类的 dp 值之和。$dp[i][j]=ps[j\%i]$。

而有了 $k\le K$，则还需要一个大小为 K 的滑动窗口，每次当 $j\ge Ki$ 时窗口下端收缩。

复杂度 $O(n^3K)$
