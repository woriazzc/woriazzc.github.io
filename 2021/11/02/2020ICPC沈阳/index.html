H. The Boomsday Project

 

题意：共享单车，每次骑要花费 rrr 元。另外还有 nnn 种租赁方式，每种方式在购买后 did_idi​ 天内有效，在有效期内能免费骑 kik_iki​ 次，购买需要花费 cic_ici​ 元。给出 mmm 条骑车记录，在第 pip_ipi​ 天骑了 qiq_iqi​ 次，问最少花费。1≤n≤500,1≤m≤105,1≤r≤109,1≤di,ki,ci≤109,0≤pi≤109,0≤qi≤3×105,∑i=1mqi≤3×1051\leq n \leq 500, 1 \leq m \leq 10^5, 1 \leq r \leq 10^9,1 \leq d_i, k_i, c_i \leq 10^9,0 \leq p_i \leq 10^9, 0 \leq q_i \leq 3 \times 10^5, \sum_{i=1}^m q_i \leq 3 \times 10^51≤n≤500,1≤m≤105,1≤r≤109,1≤di​,ki​,ci​≤109,0≤pi​≤109,0≤qi​≤3×105,∑i=1m​qi​≤3×105

dp

构造一个新的数列，连续 qiq_iqi​ 个 pip_ipi​ 代表在第 pip_ipi​ 天骑了 qiq_iqi​ 次。

每种租赁方式就相当于一种块。对于一种租赁方式，有效期一定随着当前日期单调，所以只需要维护右端点位置即可。

 

D. Journey to Un’Goro

 

题意：定义一个字符串的值等于包含奇数个 ‘r’ 的连续子串的个数，问所有只由 ‘r’，‘b’ 构成的字符串中最大的值。并输出值等于最大值的字典序最小的前100个字符串。

dfs剪枝

区间 [i,j][i,j][i,j] 中包含奇数个 ‘r’，说明 pre[i−1]pre[i-1]pre[i−1] 与 pre[j]pre[j]pre[j] 异号，pre[i]pre[i]pre[i] 表示前 iii 个字符中的 ‘r’ 的个数。

显然当 preprepre 数组中奇数值与偶数值相差不超过 1 个时最优。由于 preprepre 数组长度为 n+1n+1n+1，所以此时分别取 ⌊n+12⌋\lfloor \frac{n+1}{2}\rfloor⌊2n+1​⌋ 与 ⌈n+12⌉\lceil \frac{n+1}{2}\rceil⌈2n+1​⌉ 。

dfs时记录 preprepre 中奇数与偶数个数，当超过 ⌈n+12⌉\lceil \frac{n+1}{2}\rceil⌈2n+1​⌉ 时剪枝。
