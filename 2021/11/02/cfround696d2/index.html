C. Array Destruction

 

题意：给定一个数组。先选择一个数 x，再从数组中选择两个和为 x 的数，从数组中删去，并 x 变成这两个数中大的那个。问是否可行，以及每一步的选择。

最开始的用来合成 x 的两个数中，必定有一个是数组中最大的数，否则它以后就用不到了。所以只需要再枚举另一个数。

这样就已知 x 与两个数中大的那个数了，之后的每一步就都是可以模拟下去的了。

所以对每一次枚举，O(n)O(n)O(n) 模拟进行验证。
