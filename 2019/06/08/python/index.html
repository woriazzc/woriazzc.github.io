太久不用Python，记一下易错的点。

  • Python传参数时，兼具传值与传引用，当参数为可变数据类型，如列表时，传引用，可直接在函数内改变参数的值；当参数为不可变类型时，传值。
  • list+=与=+有差别，+=是在原列表上改动，因此=+地址会改变。
    若要在函数内改动list，建议用+=。
  • narray与list类型不同，使用时注意转换。
  • 列表[:-1]表示正向取到最后一个数，最后一个数不要；
    [::-1]表示逆序取完所有数
