数论

 

  1. 小于n且与n互质的数的和=$\frac{n\cdot\phi (n)}{2}$

  2. 原图 $(x,y) ->$ 新图 $(x+y,x-y)$ ：曼哈顿距离 -> 切比雪夫距离

    $(x,y)->(\frac{x+y}{2},\frac{x-y}{2})$，切比雪夫 -> 曼哈顿

  3. 全错位排列（每个位置都和其下标不同的情况数）：$f(n)=(n−1)(f(n−1)+f(n−2)),f(1)=0,f(2)=1$

  4. $f(i,j)=f(i-1,0)+f(i-1,1)+f(i-1,2)+\cdots +f(i-1,j)\\
    =\sum_{k=0}^{j}f(i-1,k)\\=\frac{(i+j)!}{i!\cdot j!}$

树上差分

 

 边差分：

 例如有一次操作是把红点（u）到绿点（v）之间的路径全部加x。那么我就标记dlt[u]+=x,dlt[v]+=x。然后我们要在lca(u,v)处标记dlt[lca(u,v)]-=2x。这样就使得加x的效果只局限在u..v，不会向lca(u,v)的爸爸蔓延。

 点差分：

 dlt[u]+=x,dlt[v]+=x，把dlt[lca(u,v)]-=x并把dlt[fa[lca(u,v)]]-=x。
